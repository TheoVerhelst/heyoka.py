heyoka.make_vars(*args: str) → expression | list[expression]#

Create variable expressions from strings.

This function will return one or more expression instances containing variables constructed from the input arguments. If a single argument is supplied, a single expression is returned. Otherwise, a list of expressions (one for each argument) is returned.
